# 快速开始

从零启动一个具备 AI 能力的 Bluesky 客户端，只需 5 分钟。你可以选择 **终端（TUI）** 或 **浏览器（PWA）** 两种运行模式——二者共享相同的底层逻辑，只是界面不同。

---

## 前置条件

| 依赖 | 最低版本 | 说明 |
|------|---------|------|
| **Node.js** | >= 18 | 推荐 v20 LTS |
| **pnpm** | 任意 | 包管理器，`npm install -g pnpm` 安装 |
| **终端**（TUI） | — | Windows Terminal、iTerm2、Kitty 等支持 raw mode 的终端 |
| **浏览器**（PWA） | — | Chrome / Firefox / Safari 最新版 |

[来源](README.md#L22-L27)

## 第一步：克隆与安装

```bash
git clone https://github.com/epheiamoe/bsky.git
cd bsky
pnpm install
pnpm -r build
```

这三个命令依次完成：
1. 克隆整个 monorepo 仓库
2. 安装所有子包（`@bsky/core`、`@bsky/app`、`@bsky/tui`、`@bsky/pwa`）的依赖
3. 构建所有子包的 TypeScript 源码 → 编译输出到各 `dist/` 目录，供运行时引用

[来源](README.md#L29-L35)

## 第二步：选择启动模式

### 模式 A：TUI 终端客户端

TUI 需要 `.env` 文件提供凭据。项目根目录下已有 `.env.example` 模板：

```bash
# 复制模板
cp .env.example .env
```

编辑 `.env`，填入你的 BlueSky 凭据和 LLM API Key：

```env
# Bluesky 账号（必填）
BLUESKY_HANDLE=your-handle.bsky.social
BLUESKY_APP_PASSWORD=xxxx-xxxx-xxxx-xxxx

# LLM API（OpenAI 兼容格式，推荐 DeepSeek）
LLM_API_KEY=sk-your-api-key
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-v4-flash

# AI 高级功能（可选）
LLM_THINKING_ENABLED=true
LLM_VISION_ENABLED=false

# 翻译目标语言（可选）
TRANSLATE_TARGET_LANG=zh
```

> **如何获取 Bluesky App Password？** 登录 [bsky.app/settings](https://bsky.app/settings) →  Moderation → App Passwords → 创建新密码。
>
> **LLM API Key？** 注册 [DeepSeek](https://platform.deepseek.com) 获取 API Key，或在[配置指南](配置指南.md)中查看其他供应商。

填写完成后启动：

```bash
cd packages/tui && npx tsx src/cli.ts
```

首次启动时，如果 `.env` 未配置完整，终端会自动进入 **交互式设置向导（SetupWizard）**，引导你一步步填入 Handle、密码、选择模型供应商、输入 API Key 等——完成后自动写入 `.env` 并启动主界面。[来源](packages/tui/src/cli.ts#L81-L99)

启动成功后，你会看到一个终端内的 Bluesky 全功能客户端。核心操作快捷键包括：

| 按键 | 功能 |
|------|------|
| `t` | 时间线 |
| `p` | 发帖 |
| `n` | 通知 |
| `s` | 搜索 |
| `a` | AI 对话 |
| `c` | 设置 |
| `q` / `Ctrl+C` | 退出 |

完整快捷键映射见 [TUI 快捷键与导航](tui-快捷键与导航.md)。

[来源](README.md#L37-L42) | [来源](packages/tui/src/cli.ts#L1-L14)

### 模式 B：PWA 浏览器应用

PWA **不需要 `.env`**——所有凭据通过浏览器登录表单输入，仅保存在浏览器本地存储中。

```bash
cd packages/pwa && pnpm dev
```

Vite 开发服务器会输出一个本地地址（默认 `http://localhost:5173`）。在浏览器中打开：

1. **登录**：页面自动跳转到登录表单，输入 Bluesky Handle + App Password 即可
2. **配置 AI**：点击侧栏设置图标，在 Settings 弹窗中输入 LLM API Key、Base URL 和模型名
3. 所有配置保存在 `localStorage` 中，页面刷新不会丢失

## 验证效果

TUI 和 PWA 启动后，你会看到熟悉的 Bluesky **时间线（Timeline）**——包含你关注的用户发布的帖子。在此基础上，验证几个核心 AI 功能：

- **AI 对话**：按 `a`（TUI）或点击 AI 图标（PWA），输入 `/help` 查看可用指令

## 常见问题

**Q: `pnpm install` 报错说缺少 esbuild？**
`pnpm-workspace.yaml` 中已配置 `onlyBuiltDependencies: ["esbuild", "sharp"]`，pnpm 会自动处理。如果仍有问题，尝试 `pnpm rebuild`。

**Q: TUI 启动后乱码或界面错乱？**
TUI 依赖终端的 raw mode 支持。Windows 请使用 Windows Terminal（非 cmd/powershell），macOS 使用 iTerm2 或 Kitty。不支持 Git Bash。

**Q: 如何切换 AI 模型？**
TUI 编辑 `.env` 中的 `LLM_MODEL`；PWA 在设置页面修改。支持 DeepSeek、Mistral 等多个供应商，详见[配置指南](配置指南.md)。
