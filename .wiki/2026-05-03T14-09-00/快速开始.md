## 快速开始

5 分钟内完成安装、配置，同时运行终端（TUI）和网页（PWA）客户端。如果你还不了解项目整体定位，建议先阅读 [概览](概览.md)。

---

### 前置条件

| 条件 | 说明 |
|------|------|
| **Node.js** >= 18 | [下载安装](https://nodejs.org/) |
| **pnpm** | 安装后执行 `npm install -g pnpm` |
| **Git** | [下载安装](https://git-scm.com/) |
| 终端 | TUI 需要支持 raw mode（Windows Terminal、iTerm2、Kitty 均可） |
| 浏览器 | PWA 需要现代浏览器（Chrome / Firefox / Safari） |

> 如果 `node --version` 输出低于 v18，请升级 Node.js。

[来源](../README.md#L14-L17)

---

### 第一步：获取代码

```bash
git clone https://github.com/epheiamoe/bsky.git
cd bsky
```

项目采用 **Monorepo** 结构，包含四个包：`@bsky/core`、`@bsky/app`、`@bsky/tui`、`@bsky/pwa`。依赖流向如下：

```mermaid
graph LR
    core[@bsky/core] --> app[@bsky/app]
    app --> tui[@bsky/tui]
    app --> pwa[@bsky/pwa]
```

详细架构说明见 [项目结构与包依赖](项目结构与包依赖.md)。

[来源](../README.md#L21-L22)

---

### 第二步：安装依赖

```bash
pnpm install
```

这条命令会安装根目录及所有子包的依赖。pnpm 使用 workspace 协议自动链接 `@bsky/core`、`@bsky/app` 等内部包。

[来源](../README.md#L23)

---

### 第三步：构建所有包

```bash
pnpm -r build
```

`-r` 表示对所有子包递归构建，编译顺序由依赖关系自动决定：`@bsky/core` → `@bsky/app` → `@bsky/tui` + `@bsky/pwa`。

构建产物输出到各包的 `dist/` 目录。

[来源](../README.md#L24)

---

### 运行方式一：TUI 终端界面

TUI（Text-based User Interface）在终端中运行，使用 Ink（React-on-terminal）渲染。你需要配置 Bluesky 账号和 AI 密钥。

#### 1. 复制环境变量模板

```bash
cp .env.example .env
```

#### 2. 编辑 `.env` 文件

用文本编辑器打开 `.env`，填写以下关键字段：

```env
# Bluesky 账号（必填）
BLUESKY_HANDLE=你的用户名.bsky.social
BLUESKY_APP_PASSWORD=你的应用密码

# LLM API（必填，推荐 DeepSeek）
LLM_API_KEY=sk-你的API密钥
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-v4-flash

# AI 思考模式（可选）
LLM_THINKING_ENABLED=true

# AI 视觉模式（仅对支持视觉的模型启用）
LLM_VISION_ENABLED=false

# 翻译目标语言（可选）
TRANSLATE_TARGET_LANG=zh
```

**获取 Bluesky 应用密码**：登录 [bsky.app](https://bsky.app) → 设置 → 隐私与安全 → 应用密码 → 新建密码。

**获取 API 密钥**：注册 [DeepSeek](https://platform.deepseek.com/) 或其他 OpenAI 兼容服务商，创建 API Key。

每个字段的详细说明见 [环境与凭据配置](环境与凭据配置.md)。

[来源](../.env.example#L1-L18)

#### 3. 启动 TUI

```bash
cd packages/tui
npx tsx src/cli.ts
```

首次启动会加载时间线，使用键盘操作：

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 移动光标 |
| `Enter` | 查看帖子详情 |
| `r` | 回复帖子 |
| `l` | 点赞/取消点赞 |
| `q` | 退出 |

更多快捷键见 [TUI 终端界面实现](TUI-终端界面实现.md)。

[来源](../packages/tui/package.json#L17)

---

### 运行方式二：PWA 网页界面

PWA 在浏览器中运行，**无需配置 `.env` 文件**——所有凭据通过登录表单输入，保存在浏览器 `localStorage` 中。

```bash
cd packages/pwa
pnpm dev
```

浏览器打开 `http://localhost:5173`，你会看到登录页面：

1. 输入 **Bluesky Handle**（如 `user.bsky.social`）
2. 输入 **App Password**（同上）
3. 点击登录

登录后即可浏览时间线。AI 的 API Key 可以在设置页面（⚙️ 图标）中配置。

> **在线体验**：你也可以直接访问 [ai-bsky.pages.dev](https://ai-bsky.pages.dev) 体验已部署版本。

[来源](../docs/PWA_GUIDE.md#L3-L14)
